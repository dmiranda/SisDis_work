Hundir la flota:
	- BaseDatos -> contiene un fichero .sql para la creación, y relleno de las dos tablas necesarias para el correcto funcionamiento de la aplicacion
	- servidor -> Contiene todos los ficheros necesarios para la ejecución del servidor para la aplicacion
	- cliente -> Contiene todos los ficheros necesarios para la ejecucion del cliente para la aplicacion
	
	Tanto el directorio cliente como el directorio servidor poseen un fichero makefile, con las ordenes para compilar y ejecutar la aplicacion. Para el correcto funcionamiento, se ha de modificar tanto la IP como el puerto para del rmiregistry.
